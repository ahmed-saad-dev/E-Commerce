# MERN E-Commerce Platform

A full-stack e-commerce application built with MERN Stack.

## Features

- Authentication & Authorization
- Product Management
- Shopping Cart
- Order Management
- Admin Dashboard
- Responsive Design

## Technologies

- React.js
- Redux Toolkit
- Node.js
- Express.js
- MongoDB
- JWT

## Installation

npm install
npm run dev
